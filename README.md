# 🌌 Constellation Viewer
![Python](https://img.shields.io/badge/Python-3.1-blue)
![Status](https://img.shields.io/badge/Status-OnProgress-brightgreen)
![License](https://img.shields.io/badge/License-MIT-lightgrey)

콘솔 환경에서 별자리 좌표 데이터를 기반으로 
별자리를 **시각적으로 출력**하고,  
**생일에 따른 별자리** 또는 **유명한 별자리**를 선택해 확인할 수 있는 파이썬 프로그램이다.

---

## 주요 기능

### 생일에 따른 별자리 보기
- 생일(MM DD)을 입력하면 해당 날짜의 별자리를 자동 계산
- 별자리 형태를 콘솔 ASCII 아트로 출력
- 별자리 성격 설명 + 관측 시기 및 하늘 위치 정보 제공

### 유명한 별자리 보기
- 카시오페아자리, 큰곰자리(북두칠성), 오리온자리 등
- 목록에서 선택하여 바로 시각화

### 콘솔 기반 정밀 시각화
- 모든 별 좌표는 **실수 좌표(소수점 포함)** 기반
- 좌표 스케일링을 통해 콘솔에서도 최대한 정밀한 위치 표현
- 가로·세로 비율 보정을 통해 왜곡 최소화

---

## 구현 방식

### 좌표 처리
- 별 좌표는 `(x, y)` 실수 형태로 저장
- 출력 시 좌표를 확장 및 스케일링하여 미세한 위치 차이 반영
- 콘솔 문자 비율을 고려한 가로·세로 보정 적용

### 렌더링
- 2차원 문자 그리드에 별(`✦`) 배치
- 중앙 정렬 및 자동 여백 처리
- 내부 식별자(`aries`, `famous_orion` 등)는 사용자 화면에 출력하지 않음

## 프로젝트 구조

```text
project/
│
├─ main.py              # 프로그램 진입점 및 메뉴 처리
├─ core/
│   ├─ data.py          # 별자리 좌표 및 설명 데이터
│   ├─ scaler.py        # 좌표 스케일링 로직
│   └─ renderer.py      # 콘솔 ASCII 렌더링
└─ README.md
```
---

## 실행 방법

```bash
python main.py
```
프로그램 실행 후 다음과 같은 메뉴가 표시된다.

```text
1. 생일에 따른 별자리 보기
2. 유명한 별자리 보기
3. 종료하기
선택:
```

## 출력 예시
```text
(별자리 ASCII 출력)

처녀자리

봄철 밤하늘 남쪽에서 관측되며 은하수 인근에 위치한다
----------------------------------------
```
